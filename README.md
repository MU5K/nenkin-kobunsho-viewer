# nenkin-kobunsho-viewer（年金機構 電子公文書変換ツール）

年金機構の電子公文書（ZIP配布）を、ブラウザ上でそのまま閲覧できるローカルツールです。
ZIPファイルを事前に展開する必要はありません。ドラッグ＆ドロップするだけで公文書を表示できます。

すべての処理はブラウザ内で完結し、公文書データが外部に送信されることはありません。

## 背景

年金機構から配布される電子公文書は XML + XSL の組み合わせで構成されています。しかし、現行の主要ブラウザ（Chrome / Edge）ではローカルの XSL を直接読み込めないため、そのままでは表示できません（IE モード廃止により一層困難になりました）。

本ツールは JavaScript で XSLT 変換を行い、ブラウザ上で公文書を閲覧可能にします。

## 使い方

1. `index.html` をブラウザ（Chrome / Edge / Firefox）で開く
2. 年金機構の電子公文書 ZIP ファイルをドラッグ＆ドロップ（またはファイル選択）
3. 検出された公文書の一覧から、閲覧したい文書をクリック

## 対応する公文書の種類

| 種類 | 送付時期 |
| --- | --- |
| 納入告知額・領収済額通知書 | 毎月 |
| 社会保険料額情報 | 毎月 |
| 基本保険料算出内訳書 | 年1回（1月） |
| 被保険者データ | 年1回（6月） |
| 増減内訳書 | 届出に変更があったとき |

※ 上記以外の種類も、XML + XSL ペアが含まれていれば表示できます。

## 印刷方法

1. 文書を表示した状態で「印刷」ボタンをクリック
2. 新しいタブに印刷用プレビューが開く
3. Ctrl+P で印刷ダイアログを開き、下表の設定に合わせて印刷

| 種類 | 用紙 | 余白 | 倍率 |
| --- | --- | --- | --- |
| 納入告知額・領収済額通知書 | A4 | デフォルト | — |
| 社会保険料額情報 | A3 | なし | 97% |
| 基本保険料算出内訳書 | A3 | なし | — |
| 被保険者データ | A4 | デフォルト | 95% |
| 増減内訳書 | A3 | デフォルト | — |

※ 印刷設定の自動適用は今後の課題です。

## 機能一覧

- ZIP ファイルのドラッグ＆ドロップ / ファイル選択
- ZIP を事前に展開する必要なし（ブラウザ内で直接処理）
- ネスト済み ZIP（ZIP 内の ZIP）の再帰展開
- XML + XSL ペアの自動検出（`<?xml-stylesheet?>` 参照）
- XSL 内の `import` / `include` のインライン解決
- XSLT 変換表示 / XML 生データ表示の切替
- 印刷機能（新しいタブで印刷プレビュー）

## 技術仕様

- **単一 HTML ファイル**で動作（サーバー不要）
- **外部依存なし** — ZIP 展開はブラウザ標準の DecompressionStream API を使用
- **XSLT 変換** — ブラウザ標準の `XSLTProcessor` API を使用
